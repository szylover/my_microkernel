\section{本章目标}

\begin{itemize}
  \item 理解交叉编译原理与 GCC 构建三元组（build / host / target）
  \item 定义 \texttt{i686-szy} 目标平台（OS target triplet）
  \item 在宿主机上构建 \texttt{i686-szy-gcc} + binutils 交叉编译器
  \item 配置 sysroot，使交叉编译器能找到 SZY-KERNEL 的头文件和 libc
\end{itemize}

\section{本章小结}
% ============================================================================

交叉编译工具链是移植 GCC 和 Vim 的前提。\texttt{i686-szy-gcc} 让我们能在宿主机上
编译 SZY-KERNEL 的用户态程序。下一章实现终端子系统（termios + PTY），
为 Vim 等交互式程序提供终端环境。
