\section{初始化顺序}

\codefile{src/kernel/core/kmain.c}
\begin{lstlisting}[style=cstyle]
pmm_init(mmap);        /* 1. 物理内存管理器 */
vmm_init();            /* 2. 虚拟内存 (页表) */
vmm_unmap_identity();  /* 3. 拆除 identity mapping */
kmalloc_init();        /* 4. 内核堆 — 依赖 PMM + VMM */
vma_init();            /* 5. VMA — 依赖 kmalloc */
\end{lstlisting}

依赖链：PMM $\to$ VMM $\to$ kmalloc $\to$ VMA。
VMA 在堆之后——因为 VMA 节点需要 \texttt{kmalloc} 分配。
