\section{本章目标}

\begin{itemize}
  \item 理解终端的历史演进（物理终端 → 虚拟终端 → 伪终端）
  \item 设计可插拔 \texttt{tty\_ops\_t} 接口（raw / canonical 模式可切换）
  \item 实现 termios 行规程（行缓冲、回显、特殊字符）
  \item 实现伪终端 PTY（master/slave 配对）
  \item 实现 VT100/ANSI 转义序列解析（光标/颜色/清屏）
  \item 通过 \texttt{KCONFIG\_TTY\_BACKEND} 切换（basic / pty）
\end{itemize}

\section{本章小结}
% ============================================================================

终端子系统通过可插拔 \texttt{tty\_ops\_t} 接口，支持 basic 和 PTY 两种模式。
termios 行规程和 VT100 解析为 Vim 等交互式程序提供了必要的终端环境。
